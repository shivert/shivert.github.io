Flyer Run GeoJson Mapping Tool

Flipp is the goto app for viewing weekly circulars on your phone. With thousands of flyers every week, it was very difficult to verify the accuracy of the flyer distribution zones without having any visual aid. This tool used the zip codes contained in each of the zones as well as some zip code GeoJSON data and created and plotted the polygons using the Google Maps API.

  • Developed for: Flipp Corp
  • Technologies Used: Ruby of Rails, GeoJSON, Google Maps API
